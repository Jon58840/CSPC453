CONTROLS:

Change Shape:
0: Circle (Also the starting shape)
1: Squares and Diamonds
2: Spiral
3: Sierpinski Triangles

Choosing a shape will reset that new shape to 2 iterations

Change Iterations:
Left Arrow Key: Decrease iterations (Minimum 1)
Right Arrow Key: Increase iterations (Each shape has a different set limit)

Arrow keys can be pressed for single iterations or held for rapid multiple iterations.
